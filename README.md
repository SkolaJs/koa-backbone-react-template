koa-backbone-react-template
===========================

Boilerplate for a koa-backbone-react project:
<br>
-Backbone
<br>
-Koa
<br>
-React
<br>
-Node 0.11.9+

##Up and Running:

-install node 0.11.9+
<br>
-`git clone git@github.com:tomkit/koa-backbone-react-template.git`
<br>
-`cd koa-backbone-react-template`
<br>
-`npm install`
<br>
-`node --harmony index.js`
